In a [[Fields|field]], if $ab=0$, then $a=0$ or $b=0$ (because if $a\neq 0$, then $a^{-1} ab=a^{-1}0=0\implies b=0$)
This property is not true for general [[Rings|rings]], e.g.
$$
\begin{pmatrix}
0 & 1 \\
0 & 0
\end{pmatrix}
^{2}=\begin{pmatrix}
0 & 0  \\
0 & 0
\end{pmatrix}
$$
## Definition
Let $R$ be a ring. If $a\in R$ is such that $ab=0$ for some $b\neq 0$, then $a$ is called a zero divisor

## Examples
An example is $\overline{2}\in \mathbb{Z} /6$, where $\overline{2}\cdot \overline{ 3}=\overline{0}$
