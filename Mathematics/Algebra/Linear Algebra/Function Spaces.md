## Definition
A [[Subspaces|subspace]] of a [[Vectorspaces|vectorspace]] [[Product Sets|$\mathbb{F}^{A}$]] for some [[Fields|field]] $\mathbb{F}$ is called a function space
In other words, a function space is a collection of functions with [[Image|image]] $\mathbb{F}$ and common [[Domain|domain]] $A$ which is closed under addition and multiplication by scalars
