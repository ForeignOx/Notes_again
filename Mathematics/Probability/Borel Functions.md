## Definition
A Borel function is a [[Functions|function]] from some [[Sets|set]] $\Omega$ to $[-\infty,\infty]$ in a [[Measurable Spaces|measurable spcace]] $(\Omega,\mathfrak{F})$ if for every $a\in\mathbb{R}$, the set
$$
\left\{ x\in \Omega :\middle|: f(x)\leq a \right\}\in  \mathfrak{F}
$$
Is measurable
